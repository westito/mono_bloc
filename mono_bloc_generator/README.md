# mono_bloc_generator

[![pub version](https://img.shields.io/pub/v/mono_bloc_generator?logo=dart)](https://pub.dev/packages/mono_bloc_generator)
[![Tests](https://github.com/westito/mono_bloc/actions/workflows/tests.yml/badge.svg?branch=main)](https://github.com/westito/mono_bloc/actions/workflows/tests.yml)

Code generator for [MonoBloc](https://pub.dev/packages/mono_bloc).

## Installation

```yaml
dev_dependencies:
  mono_bloc_generator: ^1.0.0
  build_runner: ^2.10.0
```

## Usage

```bash
dart run build_runner build --delete-conflicting-outputs
```

## Documentation

For complete documentation, examples, and usage guide, see the [MonoBloc package](https://pub.dev/packages/mono_bloc).
